I decided to use a framework called bootstrap made by twitter because it makes it much easier for me to make a good looking design since I am not necessarily good with CSS. I have added some components to the board.js file to make the game board styled with bootstrap. I also changed the names of the ruins because I found some icons that I could use and so I had to change the names to match my icons. I decided to make the coordinate input a mouse click since it is easier for the user and makes more sense than manually having to search and find the coordinates and then entering them in for every dig. When a major event happens in the game a toast notification will popup letting the user know when something important has happened, such as winning the game or finding all of a group of ruins. And lastly I made all of the ruins have their own image along with the cell markers allowing for a more visually appealing game and makes more sense because then you are actually finding more than just "="'s or "0"'s.

This game is also fully resposnive since bootstrap is responsive by default. (Adjusts layout based on size of screen)
